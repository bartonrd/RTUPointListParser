README – Input
This folder contains the source files that serve as the basis for generating the expected output located in the Expected Output folder.
Folder Structure
Parent Folder
│
├── Input
│   └── [This folder – contains the original input files]
│
└── Expected Output
    └── [Reference output files generated from these inputs]

Purpose
The files in this folder are the original inputs used for processing. They are typically point lists or other data files that will be parsed and transformed into the required output format.
Key Notes

All files in this folder corresponds to the single output file in the Expected Output folder.
These files are intended for testing, validation, and automation development.
Do not modify these files.
